HTML 이란 무엇인가?

Meaning of HTML in English

Hyper-text Markup Language, a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages.

한국어 HTML 의미

HTML은 간단히 말해 웹 개발 언어입니다. 웹 개발이란 인터넷 환경에서 구현되는 웹사이트 또는 웹 애플리케이션을 개발하는 작업을 말합니다. 우리가 매일 보는 웹사이트는 웹 서버와 클라이언트 단이 상호 통신하면서 출력해낸 결과물입니다. 따라서 웹 개발은 서버 사이드(백엔드)와 클라이언트 사이드(론트엔드)로 구분할 수 있다. HTML은 클라이언트 사이드 개발에 사용되는 언어입니다.


Hyper Text는 사용자가 필요한 정보를 자유롭게 찾아갈 수 있는 비순차적 텍스트를 의미합니다. 기존의 출판 문서는 작가의 의도대로 독자가 순서에 맞춰 읽는 형식이었습니다. 이를 탈피한 하이퍼텍스트 문서는 사용자의 의도대로 링크를 따라가면서 순서에 상관없이 읽을 수 있는 형태입니다.
Markup Language는 Mark(표시, 부호)를 이용하여 문서나 데이터의 구조를 표기하는 언어입니다. HTML에서의 Mark는 태그 ‘< >’ 를 의미하기 때문에 태그 부호를 이용해서 문서의 구조를 표현하는 언어를 말합니다. 두 용어의 의미를 합쳐보면, HTML이란 아래와 같이 정의할 수 있습니다.
